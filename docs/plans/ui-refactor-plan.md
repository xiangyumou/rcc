# UI 完全重构计划

## 目标
将项目 UI 从当前的硬编码样式完全重构为符合 UI.md 设计系统的实现。

## 当前问题
1. 完全没有 CSS 变量系统
2. 全部使用硬编码颜色（如 `#1e1e1e`, `#252526`, `#0e639c`）
3. 品牌色错误（使用蓝色而非规定的绿色 `#10a37f`）
4. 组件样式不符合规范
5. 无深色/浅色模式切换

## 重构步骤

### Phase 1: 创建 CSS 变量系统
**文件**: `src/client/globals.css` (新建)

内容：
1. 12 个核心变量
2. 透明度变体（-a10, -a20）
3. 圆角系统（radius-sm/md/lg/xl）
4. 间距系统（space-xs/sm/md/lg/xl）
5. Z-index 层级
6. 阴影系统
7. 深色模式变量

### Phase 2: 重写基础样式
**文件**: `src/client/style.css` (重写)

内容：
1. 使用 globals.css 的变量
2. 字体系统（Inter）
3. 排版规范
4. 可访问性支持

### Phase 3: 重写组件样式
**文件**: `src/client/style.css`

按顺序重写：
1. Button（6 种变体 + 尺寸系统）
2. Input（输入框、复选框）
3. Card（卡片容器）
4. Modal（弹窗）
5. Sidebar（侧边栏）
6. Terminal/Toolbar（终端区域）
7. File Browser（文件浏览器）

### Phase 4: 更新 HTML
**文件**: `src/client/index.html`

更新：
1. 添加 Inter 字体链接
2. 添加主题切换类
3. 更新类名以匹配新样式

## 关键变更对照表

| 元素 | 当前 | 变更后 |
|------|------|--------|
| 主背景 | `#1e1e1e` | `var(--bg)` |
| 品牌色 | `#0e639c` | `#10a37f` |
| 卡片背景 | `#252526` | `var(--surface)` |
| 边框 | `#333` | `var(--border)` |
| 文字 | `#e0e0e0` | `var(--text)` |
| 圆角 | `4px/8px` 混用 | `var(--radius)` 系统 |

## 验证清单
- [ ] 无硬编码颜色值（除变量定义外）
- [ ] 所有颜色使用 CSS 变量
- [ ] 深色/浅色模式可切换
- [ ] 按钮 6 种变体完整
- [ ] 间距符合 4/8/16/24/32 系统
- [ ] 圆角符合规范
- [ ] 包含 prefers-reduced-motion
